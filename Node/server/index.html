前期准备

  • 服务器（或者本机）

    • 我用了腾讯云最低端的windows系统（不推荐）
      • 111.229.237.10（公）172.17.0.14（内）
      • 使用windows系统需要安装openssh
        • 可以通过修改ssh配置增强服务器安全等级
          • 例如： 默认端口22 可以通过修改Port增强
          • 增加登陆用户
          • 关闭Root用户登陆、密码登陆等
      • 推荐使用zsh(Mac 自带)
        • 设置别名
